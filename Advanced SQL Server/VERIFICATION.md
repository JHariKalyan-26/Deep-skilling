# Advanced SQL Server Verification Guide

Each exercise folder contains:

- `solution.sql` — SQL code
- `README.md` — exercise explanation
- `Output/output.txt` — expected output description

Important: `Output/output.txt` is an expected-output reference.  
For actual verification, execute each `solution.sql` file in SQL Server Management Studio and save a screenshot of the Results/Messages panel inside the corresponding `Output` folder.
