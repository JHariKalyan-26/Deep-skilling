# Angular 20 - Product Management Application

Status: Completed

This module demonstrates the major Angular concepts listed in the Cognizant Deep Skilling handbook:

1. Angular CLI project setup
2. Standalone components
3. Property, event and two-way data binding
4. Built-in directives and pipes
5. Parent-child communication using inputs and outputs
6. Reactive forms and validation
7. Dependency injection and reusable services
8. Routing and navigation
9. Route guards
10. HttpClient configuration
11. RxJS observables and operators
12. Service-based state management
13. Custom pipe
14. Production build

## Run

```powershell
cd product-management
npm install
npm start
```

Open:

```text
http://localhost:4200
```

## Build

```powershell
npm run build
```
